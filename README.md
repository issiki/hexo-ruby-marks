# hexo-ruby-marks

Ruby character tag for Hexo.

## Install
via npm:

`$ npm i hexo-ruby-marks`

## Usage

**CONFLICT** with other hexo plugins that have registered `ruby` tag.

### {% ruby base|top text %}

e.g.:
```
{% ruby 初音未来|Hatsune Miku %}
```
---> `<ruby><rb>初音未来</rb><rp>(</rp><rt>chū yīn wèi lái</rt><rp>)</rp></ruby>`
